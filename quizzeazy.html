Technologies utilisées

    Frontend
    • HTML5/CSS3: Interface pure (sans JavaScript)
    • Tailwind CSS: Framework CSS utility-first
    • Forms Symfony: Formulaires dynamiques
    • Twig: Moteur de templates
    Backend
    • html/Symfony: Framework MVC
    • Doctrine ORM: Gestion base de données
    • MySQL: Base de données relationnelle

Fonctionnalités principales

  • Quiz par catégories (10 questions)
  • Affichage séquentiel avec réponses progressives
  • Historique et scores des utilisateurs
  • Création de quiz par les utilisateurs connectés
  • Inscription avec validation email
  • Authentification sécurisée
  • Modification de profil et mot de passe
  • Panel admin complet (CRUD utilisateurs/quiz)
  • Analytics et graphiques de statistiques
