Technologies utilisées

    Frontend
    • React: Framework JavaScript pour l'interface utilisateur
    • Tailwind: Styles et animations
    • React Router: Navigation entre pages
    • Fetch: Requêtes API
    Backend
    • API REST: Fournie via Docker
    • Docker: Conteneurisation de l'API
    • JSON: Format d'échange de données
    • Pagination: Gestion des grandes listes
    • Recherche: Système de filtrage multi-critères

Fonctionnalités principales

  • Listing albums : Affichage paginé avec navigation vers détails
  • Détail album : Informations complètes + liste des pistes
  • Listing artistes : Navigation par artiste avec pagination
  • Détail artiste : Albums de l'artiste sélectionné
  • Listing genres : Catégorisation musicale
  • Détail genre : Albums par genre musical
  • Recherche universelle : Albums, artistes et genres
  • Résultats paginés : Navigation fluide dans les résultats
  • Page d'accueil : Sélection aléatoire d'albums
  • Navigation : Liens croisés entre contenus
  • Connexion / CRUD
