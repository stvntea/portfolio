Technologies utilisées

    Frontend
    • React Native: Framework mobile cross-platform (iOS/Android)
    • JavaScript ES6+: Logique applicative et composants
    • React Navigation: Gestion de la navigation entre écrans
    • AsyncStorage: Stockage local des données utilisateur
    • Camera API: Accès à la caméra du smartphone
    • Tailwind CSS: Framework CSS utility-first
    • Image Picker: Sélection d'images depuis la galerie
    • State Management: Gestion d'état avec React Hooks
    Backend
    • API REST: Intégration avec l'API fournie
    • HTTP Requests: Fetch API pour les appels réseau
    • JSON: Format d'échange de données
    • Authentification JWT: Gestion sécurisée des sessions
    • Upload d'images: Envoi de fichiers multimédia
    • Expo: Plateforme de développement et déploiement

Fonctionnalités principales

    • Inscription et connexion utilisateur
    • Gestion de session sécurisée
    • Protection des pages privées
    • Prise de photo ou sélection galerie
    • Choix du destinataire et durée d'affichage
    • Upload vers l'API
    • Affichage des snaps reçus
    • Lecture temporisée avec suppression automatique
    • Synchronisation avec le serveur

Durée de production

2 semaines
